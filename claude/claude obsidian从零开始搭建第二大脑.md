从零开始搭建步骤

1. 安装obsidian和插件
 安装好后新建仓库，选择新建的仓库，设置->第三方插件->社区插件市场->BRAT插件安装好后点击启用->左侧会多一个BRAT符号点击->Plugins:Add a beta plugin for testing(with or without version)->输入https://github.com/YishenTu/claudian，选择最新版本 点击Add plugin
 安装好后点击左侧的Open claudian就可以对话了

2. 安装claude code 
  并登陆可用的claude code账号

3. 安装chrome浏览器插件
  搜索obsidian Web clipper，点击add to chrome
  安装好后chrome浏览器的插件里就会有这一项了

4. 使用obsidian里的claude对话搭建一个框架
  对话内容举一个例子
  
 我想从零搭建一个属于自己的第二大脑，你就是我的AI图书馆员。按这个流程来，1. 先采访我，你觉得需要问几个问题来了解我就问几个，至少要搞清楚我是谁，在做什么，希望这个图书馆帮我做什么，希望你怎么跟我说话。2.问完之后，写一份整理运营手册(CLAUDE.md)放在根目录。3.根据我的项目和目标，建立你认为最合适的几个馆藏分区，告诉我你的建议和理由，我可以接受或调整。4、最后，帮我开一个今天的daily笔记，记录一下我们今天做了什么。 几点硬约束：只能用我接下来回答你的内容来写； 不要引用你已知的其他关于我的信息。


然后回答claude的几个问题，就能建好笔记目录了

最后还可以写一个memory.md
在图书馆里再建一个文件叫memory.md留空，然后在整管手册末尾加一段，告诉你下次再来上班时发现我的新偏好新习惯，主动写到memory.md中


5. 使用
  例如从claude对话框中保存到obsidian,在chrome浏览器中打开claude对话,选择obsidian web clipper插件，点击Add to obsidian，就可以导入到obsidian


是不是操作起来很简单，开始愉快的玩耍吧
